# InvoLead Enterprise Frontend

Next.js + TypeScript enterprise starter with App Router, Tailwind, TanStack Query, ShadCN-style components, and production tooling.

Quick start:

```bash
npm install
npm run dev
```

Lint, format, typecheck:

```bash
npm run lint
npm run format
npm run type-check
```

Notes:
- Run `npx husky install` after `npm install` to enable commit hooks.
- Fill `.env.local` from `.env.example` before running in production.
